# Encriptador de Texto

## Descripción

Este proyecto es una aplicación web sencilla que permite encriptar y desencriptar texto. La interfaz es intuitiva, diseñada para facilitar el uso por parte del usuario, con un enfoque en la simplicidad y funcionalidad. El proyecto está compuesto por tres archivos principales: `index.html`, `style.css`, y `script.js`.

## Estructura del Proyecto

- **index.html**: Contiene la estructura básica de la página web utilizando HTML. Aquí se definen las áreas donde el usuario puede ingresar texto, encriptarlo, desencriptarlo y copiar el resultado.

- **style.css**: Proporciona el estilo visual de la página. Incluye configuraciones de fuentes, colores, y la disposición de los elementos en la interfaz.

- **script.js**: (No proporcionado en la descripción) Debería contener la lógica de encriptación y desencriptación, así como el manejo de eventos para los botones.

## Funcionalidades

- **Encriptar**: Toma el texto ingresado por el usuario y lo transforma según un algoritmo de encriptación.
- **Desencriptar**: Reversa el proceso de encriptación, devolviendo el texto original.
- **Copiar**: Permite copiar el texto encriptado o desencriptado al portapapeles del usuario.

## Estructura HTML

La página HTML se organiza en varias secciones:

- **Header**: Contiene un logo que representa la aplicación.
- **Main**: La sección principal incluye un área de texto donde el usuario ingresa el texto para encriptar o desencriptar, botones para realizar estas acciones, y un área para mostrar el mensaje resultante.
- **Footer**: Incluye información sobre el autor del proyecto.

## Estilos CSS

El archivo CSS utiliza:

- **Fuentes**: Se utiliza la fuente 'Inter', importada desde Google Fonts.
- **Colores**: Paleta de colores suaves y femeninos, con énfasis en tonos morados y rosados.
- **Disposición**: Uso de `flexbox` para organizar los elementos de la página.
- **Botones**: Estilo personalizado para los botones de encriptar, desencriptar, y copiar, con colores y bordes que coinciden con el tema visual de la página.

## Cómo Usar

1. **Ingresar texto**: Escriba el texto que desea encriptar en la primera área de texto. Recuerde que solo se aceptan letras minúsculas y sin acentos.
2. **Encriptar/Desencriptar**: Haga clic en el botón correspondiente para realizar la acción deseada.
3. **Copiar**: Una vez obtenido el texto encriptado o desencriptado, utilice el botón de copiar para guardarlo en su portapapeles.
